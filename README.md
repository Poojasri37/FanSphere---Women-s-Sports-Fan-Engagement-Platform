FanSphere is a modern sports-tech web application designed to enhance fan engagement for women's sports, with a focus on cricket. It provides an immersive, gamified experience for fans to follow live matches, participate in challenges, and connect with a community.

Features
Section	Description
Home	Welcome dashboard with upcoming matches, quick stats, and featured athletes
Live Match	Real-time scores, interactive polls, predictions, reactions, and live chat
Fan Arena	Gamified challenges, points system, leaderboards, and achievement badges
Personalize	Customize favorite teams, players, sports, and notification preferences
Insights	AI-powered match insights, player stats, and performance analytics
Community	Moderated discussion space, match chat rooms, and trending topics
Profile	Activity summary, earned badges, and accessibility settings
Tech Stack
React 18 + TypeScript
Vite (build tool)
Tailwind CSS + shadcn/ui components
React Router for navigation
Lucide React for icons
Recharts for data visualization
Design System
Font: Outfit (modern sans-serif)
Primary: Teal/Cyan (hsl(174, 72%, 40%))
Accent: Coral/Salmon (hsl(12, 80%, 62%))
Gold: Achievement highlights (hsl(45, 85%, 55%))
Light theme with professional sports-tech aesthetic
